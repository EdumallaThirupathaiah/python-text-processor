Parallel Text Handling Processor

Project Description This project is a scalable text processing system that performs sentiment analysis on large datasets using parallel processing. It supports CSV, TXT files, and manual input, and displays results in an interactive dashboard.

Features

Upload CSV and TXT files
Manual text input
Parallel text processing
Sentiment analysis (Positive / Negative / Neutral)
Dashboard with results and insights
Email export feature

Tech Stack

Python
Streamlit
SQLite

How to Run
pip install -r requirements.txt
streamlit run app.py
